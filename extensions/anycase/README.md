# AnyCase

Convert the casing of any selected text without leaving your keyboard.

Select text in any app, trigger AnyCase, pick a format — the converted text is pasted back instantly. Works in any text field: code editors, terminals, notes, browsers, anywhere.

## Commands

| Command | What it does |
|---|---|
| **Change Casing** | Open a searchable list of all formats with a live preview of your selected text |
| **To UPPERCASE** | LIKE THIS |
| **To lowercase** | like this |
| **To Title Case** | Like This |
| **To Sentence case** | Like this |
| **To camelCase** | likeThis |
| **To PascalCase** | LikeThis |
| **To snake_case** | like_this |
| **To kebab-case** | like-this |
| **To SCREAMING_SNAKE_CASE** | LIKE_THIS |
| **To dot.case** | like.this |

## Usage

1. Select text in any app
2. Trigger AnyCase (e.g. `⌥Space`)
3. Either search for a specific format and press `↵`, or use one of the direct commands

The converted text is pasted directly back into the focused input, so **`⌘Z` undo works natively** in the receiving app.

## Tips

- Assign a hotkey (e.g. `⌥Space`) to **Change Casing** in Raycast → Extensions → AnyCase
- Use direct commands (e.g. "To snake_case") for one-shot conversions with no UI
- Smart word splitting handles spaces, underscores, hyphens, dots, and camelCase/PascalCase boundaries — so `myVariableName` → `my-variable-name` correctly
